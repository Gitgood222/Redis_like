       设计一个用 C++ 从零实现的类 Redis 内存数据库项目的完整方案，用于面试展示。
    
       项目目标
       - 2-3 周完成
       - 展示 C++ 功底和系统设计能力
       - 面试官容易理解和提问
    
       核心需求
       1. RESP 协议兼容（可使用 redis-cli 直接连接操作）
       2. 支持 5 种核心数据结构：String、Hash、List、Set、Sorted Set
       3. 持久化：RDB 快照 + AOF 日志
       4. 过期策略：惰性删除 + 定期删除
       5. 基于 epoll 的事件驱动网络层
       6. 单线程事件循环（与 Redis 一致）
    
       设计要求
    
       - 现代 C++17/20，充分利用 RAII、智能指针、move 语义、模板
       - CMake 构建系统
       - 模块化设计，每个模块职责清晰
       - 提供压测对比数据（与 Redis 对比）
       - README 完整
